# FinAgent Pipeline Plan

## Current Architecture

### Parallel Pipeline Structure
```
Stock Tickets (1-5)
    │
    └── ThreadPoolExecutor(max_workers=5) ─── Outer Parallelism
            │
            ├── AAPL Pipeline ──────────────────────────────────────────────
            │       │
            │       └── ThreadPoolExecutor(max_workers=7) ─── Inner Parallel
            │               │
            │               ├── Step 1: Fundamentals (yfinance)
            │               ├── Step 2: Sentiment & Social (yfinance + Reddit)
            │               ├── Step 3: Technical (yfinance)
            │               ├── Step 4: Market Overview (yfinance)
            │               ├── Step 5: Global Economy (World Bank API)
            │               ├── Step 6: Fund Holdings (agnes-2.0-flash)
            │               └── Step 7: Past Lessons (Qdrant DB)
            │                       │
            │                       └── Wait all complete
            │                               │
            │                               └── ThreadPoolExecutor(max_workers=2)
            │                                       │
            │                                       ├── Step 8.1: Bull (xiaomi/mimo-v2.5)
            │                                       └── Step 8.2: Bear (minimax/minimax-m3)
            │                                               │
            │                                               └── Wait both complete
            │                                                       │
            │                                                       └── Step 8.3: Debate (z-ai/glm-5.2)
            │                                                               │
            │                                                               └── Step 9: Trading (z-ai/glm-5.2)
            │                                                                       │
            │                                                                       └── Step 10: Lesson (agnes-2.0-flash, background)
            │
            ├── GOOG Pipeline (same structure)
            │
            └── MSFT Pipeline (same structure)
```

## Exception Handling
- Each step wrapped in try/except
- Failed steps return error message but don't stop pipeline
- Steps 8.1/8.2 failures skip 8.3 and 9
- Step 8.3 failure skips step 9
- Timeout per step: 180 seconds
- Connection retry: 2 attempts with exponential backoff
